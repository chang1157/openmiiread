# openmiiread
书迷
- 书迷一开始作为一款开发者自用的小说阅读软件，fork自kunfei，基于个人喜好定制了ui和其他功能和逻辑。随着书友们的增多，收集到不少的反馈和建议，也收获了很多的支持与好评，谢谢大家。
